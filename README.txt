Fera Digital Strategy - One Page Site

Contenuto:
- index.html
- styles.css
- script.js
- cartella assets con i file logo già inclusi

Cose da cambiare subito:
1. Sostituisci tutti i link Calendly:
   cerca "https://calendly.com/INSERISCI-LINK" e metti il tuo link vero.

2. Sostituisci il link Instagram:
   cerca "https://instagram.com/tuoprofilo" e metti il tuo profilo reale.

3. Personalizza la sezione Collaborazioni:
   cambia testi e immagini nelle 3 card progetto.

Pubblicazione:
- GitHub: carica tutti i file nella root del repository.
- Cloudflare Pages: collega il repo GitHub e pubblica.
- Build command: nessuno
- Output directory: /

Il sito è statico, quindi non serve framework.
